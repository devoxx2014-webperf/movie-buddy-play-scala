Movie buddy
=====================================

A movie buddy application for Devoxx 2014 - Play scala

Install movie_buddy project
===========================

Clone this project

`git clone git@github.com:pepite/movie-buddy-play-scala`

Install Play or SBT
===================

`brew install play`

or

'brew install sbt'

Run
====

`play run`

or 

`sbt run`

Point your browser to http://localhost:9000/
